# AI_navigation_for_visually_impaired
YOLO 기반 시각장애인 전용 내비게이션 - 객체 인식 모델 학습 및 경량화 파트

## 1. 프로젝트 개요

Nuni는 시각장애인의 안전한 독립 보행을 지원하기 위한 AI 음성 비서 기반 내비게이션 서비스입니다.

본 프로젝트는 YOLO 기반 실시간 장애물 탐지, 점자블록 인식, 음성 안내, 긴급 요청 기능을 결합하여 시각장애인이 보행 중 마주할 수 있는 위험 요소를 인식하고 음성으로 안내하는 것을 목표로 하였습니다.

---

## 2. 문제 정의

시각장애인은 보행 중 점자블록 훼손, 장애물 방치, 보행 보조 시설 부족 등으로 인해 독립적인 이동에 어려움을 겪습니다.

기존 내비게이션 서비스는 지도 기반 경로 안내에는 강점이 있지만, 보행 중 실시간으로 변화하는 주변 환경을 인식하고 안내하는 데에는 한계가 있습니다.

이에 본 프로젝트에서는 시각장애인의 보행 안전성을 높이기 위해 다음 기능을 구현하고자 하였습니다.

- 실시간 장애물 탐지
- 점자블록 및 훼손 점자블록 인식
- 음성 기반 안내
- 긴급 상황 요청
- 관제 시스템을 통한 장애물 정보 확인

---

## 3. 주요 기능

### 실시간 장애물 탐지

YOLO 기반 객체 탐지 모델을 활용하여 보행 중 마주할 수 있는 장애물을 실시간으로 탐지하였습니다.

탐지 대상은 다음과 같습니다.

- 사람
- 자동차
- 오토바이
- 자전거
- 킥보드
- 신호등

### 점자블록 인식

점자블록은 카메라 각도와 거리, 보도 상태에 따라 형태가 다양하게 나타나기 때문에 단순 바운딩 박스가 아닌 segmentation 방식을 적용하였습니다.

이를 통해 일반 점자블록과 훼손 점자블록의 영역을 보다 정밀하게 인식하고자 하였습니다.

### 음성 안내

시각장애인이 화면을 직접 확인하기 어렵다는 점을 고려하여, 탐지 결과를 음성으로 안내하는 구조를 설계하였습니다.

### 긴급 요청 기능

보행 중 위험 상황이 발생했을 때 보호자 또는 관제 시스템에 긴급 요청을 보낼 수 있도록 하였습니다.

### 관제 시스템

탐지된 장애물 정보와 사용자 상태를 확인할 수 있는 관제 화면을 구성하였습니다.
